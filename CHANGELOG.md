# 桌面机器人 - 版本记录

## v2.0 - 2026-03-23

### 新增
- **电位器音量控制**：MCP3008 ADC + 电位器，实时调音量
  - 滑动平均滤波，变化 ≥3% 才触发
  - 端点补偿，确保 0%~100% 覆盖
  - 底部水平音量条显示（绿色填充 + 百分比）
- **PS2 摇杆表情控制**：上下左右切表情
  - ⬆️ happy / ⬇️ sad / ⬅️ angry / ➡️ love / 🎯 surprised
  - 按键走 GPIO12 内部上拉，方向走 MCP3008 CH1/CH2
- **SPI 启用**：`dtparam=spi=on`，使用 `spidev0.0` (CE0)
- 删除屏幕音量 +/- 按钮，退出按钮移至右下角

### 修改
- 移除屏幕音量按钮，音量完全由电位器控制
- 音量条从竖向改为水平，位置在摄像头按钮右侧
- 退出按钮从中间移至最右下角

## v1.0 - 2026-03-22

### 核心功能
- OV5647 摄像头：人脸检测+跟踪
- 7 寸屏幕：左半屏信息面板 + 右半屏表情脸（8 种表情）
- HC-SR04 超声波距离感应
- PIR 人体感应自动唤醒
- PWM 舵机云台
- 语音对话：按住按钮/按键录音 → Whisper STT → OpenClaw → TTS
- 网易云音乐集成 + 屏幕播放器 Widget
- 语音-音乐联动（录音自动暂停/续播）
- HA 智能家居集成
- 桌面启动器
